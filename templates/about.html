This is a project made using Flask(Python), HTML, CSS, BOOTSTRAP and some JavaScript for CSE 3001 Software Engineering Course. We have used Google API to get data of the web searches made and procesed this data using python in the backend and display them as trends (graphs) for a easy and visual understanding to the user. HTML and CSS adre used to structure and design the website in an eye-pleasing manner. We also created a region/country based trend searcher which allows you to know about the hottest topics being searched about according to the country. There is also a search option which allows you to search for a specific keyword and returns that keywords trend and it also allows you to see the sentiment of that keyword by scraping it from top 10 google URLs

This is a project made for software engineering by Rohan Sai, Ayan Chandhra, Amit Kumar
